# 🔖 Savelt

Savelt is a mobile application built with **React Native** that allows users to save, manage, and access important links in a simple and practical way.

The app works as a **personal link storage tool**, making it easier to manually organize and quickly access saved URLs.

---

## 🚀 Features

- ✅ Manually save links
- 🗑️ Delete saved links
- 🔗 Redirect to stored links
- 📱 Simple and intuitive interface
- ⚡ Fast navigation

---

## 🛠️ Technologies Used

- React Native
- Expo
- TypeScript
- Expo Router

---

## 📱 How It Works

Users can:

1. Manually add a new link
2. View a list of saved links
3. Remove links whenever they want
4. Tap a link to open it in the browser

---

## ▶️ Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/savelt.git
```

### 2️⃣ Navigate to the project folder

```bash
cd savelt
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Start the project

```bash
npx expo start
```
